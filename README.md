# 金宫味业数字营销数据中台

**金宫味业**调味品行业全域数字营销数据中台，实现从数据采集、智能营销到可视化决策的闭环。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 原生微信小程序 | 原生框架，最佳兼容性 |
| UI | 自定义组件 | Canvas 2D 图表（折线/饼图/柱状图） |
| 后端 | Python FastAPI | 高性能异步 Web 框架 |
| 数据库 | SQLite（开发）/ MySQL（生产） | 开发阶段使用 SQLite 快速原型 |
| 爬虫 | Playwright（规划）/ Mock | 模拟数据 + 真实爬虫方案预留 |
| AI | DeepSeek API | 情感分析 + 智能报告生成 |

## AI 集成说明

本项目集成了 **DeepSeek** API 提供以下 AI 能力：

1. **情感分析**：对爬取的社交媒体内容自动进行情感倾向标记（正面/负面/中性）和内容类型分类
2. **智能报告**：基于实时数据自动生成日报/周报，包含数据洞察和行动建议
3. **归因分析**：简单规则引擎计算各平台对销售转化的贡献权重

> 使用 Claude Code/OpenCode 辅助完成了：
> - 项目架构设计
> - 数据模型设计
> - API 接口代码生成
> - Canvas 图表组件开发
> - 代码审查与重构

## 快速开始

### 1. 启动后端

```bash
cd backend
pip install -r requirements.txt
set DEEPSEEK_API_KEY=sk-your-api-key   # 可选，不设置则使用模板报告
python main.py
```

后端默认运行在 http://localhost:8000

### 2. 打开微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `jingong-marketing/` 目录
3. AppID 使用测试号即可
4. 在详情 → 本地设置中，**取消勾选**「不校验合法域名」

### 3. API 接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/dashboard` | GET | 获取战情室数据 |
| `/api/report/generate` | POST | 生成 AI 报告 |
| `/api/report/history` | GET | 报告历史列表 |
| `/api/monitor/competitors` | GET | 竞品监控数据 |
| `/api/monitor/sentiment` | GET | 舆情情感数据 |
| `/api/monitor/alerts` | GET | 预警信息列表 |
| `/api/monitor/tasks` | GET | 爬取任务列表 |
| `/api/monitor/scrape/{platform}` | POST | 手动触发爬取 |

## 真实场景接入说明

当前使用 Mock 数据演示，实际生产环境可：

- **抖音数据**：使用 Playwright 爬取热榜，或接入抖音开放平台 API
- **小红书**：接入小红书开放平台或使用授权数据接口
- **天猫/京东**：接入生意参谋/京东商智 API
- **竞品数据**：使用蝉妈妈、灰豚数据等第三方平台 API
